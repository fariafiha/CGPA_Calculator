# 🎓 CGPA Calculator

A Java-based University CGPA Calculator that helps students track and calculate their academic performance semester by semester.

---

## ✨ Features

- 📚 **Semester-wise GPA calculation** — Add courses for each semester
- ➕ **Course management** — Add and remove courses dynamically
- 🔢 **Grade point system** — Supports standard grading scale
- 📊 **Cumulative GPA tracking** — Calculates overall CGPA across all semesters
- ⚡ **Fast & lightweight** — Runs on any system with Java installed

## 🚀 How to Run

### Prerequisites
- Java JDK 8 or higher installed

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/fariafiha/CGPA_Calculator.git

# 2. Navigate to the project directory
cd CGPA_Calculator

# 3. Compile the Java files
javac Main.java Course.java

# 4. Run the program
java Main
```
